# 🏀 NBA Muse

NBA Muse is a web app that lets you explore NBA stats with simple natural-language queries.  
The goal of this project is to build a **StatMuse-style tool** where users can type basketball questions and get results in a clean, styled format.

---

## Features (current)
- Natural-language style search bar
- Results displayed in styled tables
- Team colors applied to outputs

---

## Tech Stack
- Next.js (App Router)
- TypeScript
- TailwindCSS
- DuckDB (local analytics)
- OpenAI API (query interpretation)

---

```bash
# Install dependencies
